# 🕷️ Data Spider - 通用数据采集工具

一个功能完善的 Python 数据采集框架，支持多目标网站并行抓取。

## ✨ 功能特性

- 多目标网站并行抓取
- 内置代理池轮换机制
- User-Agent 随机化
- Cookie 自动管理
- 定时任务调度（cron 表达式）
- 日志记录 + 异常告警
- Docker 一键部署

## 🚀 快速开始

```bash
pip install -r requirements.txt
python main.py
```

## 📦 技术栈

- Python 3.10+
- requests / BeautifulSoup4
- apscheduler（定时任务）
- Docker
